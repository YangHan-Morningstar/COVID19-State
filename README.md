# 新冠疫情数据记录员

## 一、简介

“新冠疫情数据记录员”是一款基于SwiftUI框架的新冠疫情信息App，它能够允许用户查询自己想了解的国家的疫情现状以及世界疫情现状。

## 二、系统说明

### 2.1 功能介绍

* 国家疫情信息：用户输入国家名称，系统会自动在界面上更新数据。
* 世界疫情信息：系统直接在界面显示世界的疫情信息。
